# dbc2code — DBC自动代码生成工具
Generate C files from DBC file。

## 1. 克隆仓库
```bash
git clone https://github.com/JacksonCui-Longji/Jackson-Windows-Lab.git
cd Jackson-Windows-Lab/dbc2code
```
## 2. 新建一个虚拟环境（可选但推荐）
```bash
python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # Linux/Mac
```
## 3. 安装依赖
```bash
pip install -r requirements.txt
```
## 4. 运行工具
```bash
python generator.py
```
